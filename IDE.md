# PROJECT IDEA
## Public Debate Ecosystem — TikTok Live + Knowledge Dashboard

---

# CORE VISION

Membangun sebuah ekosistem debat publik berbasis TikTok Live yang:
- populis,
- intelektual,
- semi santai,
- terbuka terhadap semua perspektif,
- dan mampu mengubah konflik gagasan menjadi pembelajaran publik.

Platform ini bukan sekadar live debat.

Tetapi:
> media alternatif digital berbasis diskusi, pengetahuan, dan partisipasi massa.

---

# CORE CONCEPT

## “Chaos → Understanding”

Mayoritas debat online:
- penuh emosi,
- tribal,
- toxic,
- dan cepat hilang.

Sistem ini dirancang untuk:
- mengarsipkan,
- merangkum,
- memvisualisasikan,
- dan mengubah debat menjadi pengetahuan kolektif.

---

## Tujuan Utama
- Edukasi publik

# LIVE STRUCTURE

## TOTAL SESSION
Platform dibagi menjadi 3 sesi utama.

---

# SESSION 1 — VIEWER SESSION

## Purpose
Membangun engagement dan energi live.

## Character
- ramai
- santai
- cepat
- provokatif

## Activities
- typing challenge
- polling
- spam challenge
- viewer attack
- komentar panas
- quick response
- audience provocation

## Goal
Meningkatkan:
- retention,
- comment rate,
- emotional engagement,
- dan participation.

---

# SESSION 2 — EDUCATION SESSION

## Purpose
Mengubah debat menjadi pembelajaran publik.

## Character

- mendalam
- argumentatif
- berbasis data

## Activities
- opening statement
- pemaparan data
- logical analysis
- AI summary
- crossfire
- audience challenge
